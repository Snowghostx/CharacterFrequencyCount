# Chinese Character Frequency Count
The purpose of this program was to take a subtitle file (vtt,srt,txt) and list every
chinese character separately with a corresponding count for each appearance it makes.
# Author's Note
This program has features cut down because I've learned that there are much better
frameworks to accomplish the end goal I wanted for this project. Despite this shift, the
use this program provides is still helpful in prioritization of characters to learn.
## Removed Features
* InputStudyPanel
* TableStudyPanel
* CCharacter
